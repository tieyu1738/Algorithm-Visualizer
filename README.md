# Algorithm Visualizer 🎨

一个交互式网页应用，用于可视化经典算法（排序和路径查找）的执行过程。通过逐步动画帮助你理解算法的工作原理。

## ✨ 特性
- **排序算法**：冒泡排序、快速排序的逐步可视化
- **路径查找算法**：Dijkstra算法在网格上的寻路演示
- **交互控制**：播放/暂停、重置、速度调节
- **响应式设计**：适配桌面和移动设备
- **教育友好**：颜色标记帮助理解算法状态
